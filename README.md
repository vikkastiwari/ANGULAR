# ANGULAR
### Command
  #### Angular CLI
     npm install -g @angular/cli@latest
  #### Update npm
    npm install -g npm
  #### App
    ng new app-name
    
### Package
  #### Bootstrap
    npm install --save bootstrap
  
### Configuration
  #### Add Bootstrap in `angular.json`
    "styles": [
                "node_modules/bootstrap/dist/css/bootstrap.min.css",
                "src/styles.css"
              ],
